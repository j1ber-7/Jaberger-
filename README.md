# 🍔 مطعم جابرجر - Jaberger

مرحباً بك! هذا مشروع **Full-Stack Web Development** لمطعم برجر باسم "جابرجر".

## 🚀 المميزات
- واجهة مستخدم عصرية (HTML/CSS).
- نظام طلبات تفاعلي (JavaScript).
- سيرفر خلفي لمعالجة البيانات (Node.js & Express).

## 📁 تقسيم المشروع
- `public/`: يحتوي على ملفات الواجهة الأمامية.
- `server.js`: هو الملف الأساسي لتشغيل السيرفر.
- `package.json`: يحتوي على الإعدادات والمكتبات المستخدمة.

## 🛠️ التشغيل
1. قم بتحميل المستودع.
2. شغل الأمر `npm install` لتثبيت الإضافات.
3. شغل السيرفر عبر `node server.js`.
